# 项目总结 (Project Summary)

## 项目概述

个人财务管理系统 (Personal Finance Manager) 是一个全栈Web应用，旨在帮助学生和自由职业者跟踪财务交易、管理预算和可视化财务数据。该项目灵感来源于Mint和YNAB等成熟的财务管理工具。

## 技术架构

### 后端技术栈
- **Node.js** + **Express.js**: 服务器框架
- **PostgreSQL**: 关系型数据库，提供强大的数据管理功能
- **JWT**: 用户认证和授权
- **Bcrypt**: 密码加密
- **Joi**: 输入数据验证
- **Helmet**: 安全头部中间件
- **CORS**: 跨域资源共享
- **Rate Limiting**: API请求频率限制

### 前端技术栈
- **React.js**: 用户界面构建
- **React Router**: 单页应用路由
- **Tailwind CSS**: 实用优先的CSS框架
- **React Hook Form**: 表单处理和验证
- **Axios**: HTTP客户端
- **React Hot Toast**: 通知组件
- **Lucide React**: 图标库
- **Chart.js**: 数据可视化（计划中）

### 数据库设计
项目采用关系型数据库设计，包含以下主要数据表：

1. **users** - 用户账户信息
2. **accounts** - 财务账户（支票、储蓄、信用卡等）
3. **categories** - 交易分类
4. **transactions** - 交易记录
5. **budgets** - 预算管理
6. **recurring_transactions** - 循环交易

## 安装和运行

### 环境要求
- Node.js 14+
- PostgreSQL 12+
- npm

### 快速开始
1. 克隆项目并进入目录
2. 运行 `npm run install-all` 安装依赖
3. 配置PostgreSQL数据库连接
4. 运行 `npm run db:migrate` 执行数据库迁移
5. 运行 `npm run dev` 启动开发服务器

### 访问地址
- 前端: http://localhost:3000
- 后端API: http://localhost:3001/api
