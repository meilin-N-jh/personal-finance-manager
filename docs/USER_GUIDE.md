# User Guide

## Overview

The Personal Finance Manager is a comprehensive web application designed to help individuals track their financial transactions, manage budgets, and visualize their spending patterns. This guide will help you navigate and use all the features effectively.

## Getting Started

### Creating an Account

1. **Navigate to the registration page**: Visit the application URL and click "Create a new account"
2. **Fill in your information**:
   - Username (unique identifier)
   - Email address
   - Password (minimum 6 characters)
   - First and Last name (optional)
3. **Click "Create account"**: You'll be automatically logged in and redirected to the dashboard

### Logging In

1. **Enter your credentials**: Use your username or email and password
2. **Click "Sign in"**: You'll be redirected to your dashboard
3. **Remember me**: Check this option to stay logged in on the same device

## Main Features

### Dashboard

The dashboard provides an overview of your financial health:

**Key Metrics**:
- **Total Balance**: Sum of all your account balances
- **Monthly Income**: Total income for the current month
- **Monthly Expenses**: Total expenses for the current month
- **Budget Progress**: How well you're sticking to your budget

**Visualizations**:
- **Spending Overview**: Charts showing your spending patterns
- **Recent Transactions**: Latest financial activities

### Accounts

Manage your financial accounts here:

**Account Types**:
- Checking accounts
- Savings accounts
- Credit cards
- Cash
- Investment accounts

**Account Features**:
- Track current balance
- Monitor account activity
- Categorize transactions by account

### Transactions

Record and manage all your financial transactions:

**Transaction Types**:
- **Income**: Money received (salary, gifts, refunds, etc.)
- **Expenses**: Money spent (bills, groceries, entertainment, etc.)

**Transaction Details**:
- Amount and date
- Description/memo
- Category classification
- Account association

**Transaction Management**:
- Add new transactions
- Edit existing entries
- Delete incorrect records
- Search and filter transactions

### Categories

Organize your transactions with custom categories:

**Default Categories**:
- **Income**: Salary, Freelance, Investment Returns, Other Income
- **Expenses**: Housing, Transportation, Food, Entertainment, Healthcare, Shopping, Bills, Other Expenses

**Custom Categories**:
- Create personalized categories
- Assign colors and icons
- Organize by income or expense type

### Budgets

Set and track financial goals:

**Budget Types**:
- **Monthly**: Budgets that reset each month
- **Yearly**: Annual budget targets

**Budget Tracking**:
- Set spending limits by category
- Monitor progress visually
- Receive alerts when approaching limits
- Track budget compliance over time

### Profile

Manage your account settings and preferences:

**Account Information**:
- View personal details
- Update contact information
- Change password
- Account security settings

## Best Practices

### Data Entry

1. **Regular Updates**: Enter transactions as they happen
2. **Accurate Categories**: Choose the most appropriate category
3. **Detailed Descriptions**: Add helpful notes for future reference
4. **Consistent Dates**: Use actual transaction dates

### Budget Management

1. **Realistic Goals**: Set achievable budget limits
2. **Regular Reviews**: Adjust budgets as needed
3. **Track Progress**: Monitor spending against budgets regularly
4. **Category Planning**: Allocate funds to different expense categories

### Security

1. **Strong Password**: Use unique, complex passwords
2. **Regular Logout**: Sign out when using shared devices
3. **Secure Connection**: Only access on trusted networks

## Troubleshooting

### Common Issues

**Transaction Not Saving**:
- Check all required fields are filled
- Verify amount is a valid number
- Ensure date is in correct format

**Budget Not Updating**:
- Refresh the page
- Check budget date range is current
- Verify category assignments

**Login Problems**:
- Confirm username/email is correct
- Check password spelling
- Use password reset if needed

### Data Accuracy

**Regular Reconciliation**:
- Compare with bank statements
- Verify account balances
- Check for duplicate entries

**Backup Data**:
- Regular data exports
- Save important records
- Document financial goals
