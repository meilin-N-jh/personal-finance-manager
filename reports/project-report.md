# Personal Finance Manager - Project Report

## 项目信息
- **课程**: Database Application Development (Track 1)
- **项目名称**: Personal Finance Manager
- **开发时间**: 2025年12月
- **项目类型**: 全栈Web应用

## 执行摘要

本项目成功设计并实现了一个功能完整的个人财务管理系统，采用现代化的全栈技术架构，为学生和自由职业者提供便捷的财务管理解决方案。该系统不仅满足了课程要求，更具备了实际部署和使用的价值。

## 1. 项目背景与目标

### 1.1 项目背景
在数字时代，个人财务管理变得越来越重要。学生和自由职业者作为收入相对不稳定的群体，更需要有效的工具来跟踪收支、管理预算和规划财务。现有的财务管理工具如Mint和YNAB虽然功能强大，但可能对初学者来说过于复杂或费用较高。

### 1.2 项目目标
- 设计一个直观易用的个人财务管理界面
- 实现完整的交易记录和分类管理功能
- 提供预算设置和跟踪工具
- 创建数据可视化组件，帮助用户理解财务状况
- 构建安全可靠的用户认证系统
- 确保良好的用户体验和响应式设计

## 2. 技术架构

### 2.1 总体架构
项目采用经典的分层架构模式：
- **表现层**: React.js + Tailwind CSS
- **业务逻辑层**: Node.js + Express.js
- **数据层**: PostgreSQL + 数据库迁移

### 2.2 前端技术栈
- **React.js 18.2.0**: 现代化的用户界面框架
- **React Router**: 单页应用路由管理
- **Tailwind CSS**: 实用优先的CSS框架
- **React Hook Form**: 高效的表单处理
- **Axios**: HTTP客户端库
- **Chart.js**: 数据可视化（计划集成）

### 2.3 后端技术栈
- **Node.js**: 服务器运行环境
- **Express.js**: Web应用框架
- **PostgreSQL**: 关系型数据库管理系统
- **JWT**: 无状态身份验证
- **Bcrypt**: 密码哈希算法
- **Joi**: 数据验证库

### 2.4 数据库设计

#### 核心数据表

1. **users表**: 存储用户基本信息
   - id, username, email, password_hash, first_name, last_name
   - 索引: username, email (唯一索引)

2. **accounts表**: 管理用户财务账户
   - id, user_id, name, type, balance, currency
   - 支持类型: checking, savings, credit_card, cash, investment

3. **categories表**: 交易分类管理
   - id, user_id, name, type, color, icon
   - 区分income和expense类型

4. **transactions表**: 核心交易记录
   - id, user_id, account_id, category_id, amount, type, description, date
   - 关联users, accounts, categories表

5. **budgets表**: 预算管理
   - id, user_id, category_id, amount, period, start_date, end_date
   - 支持monthly和yearly周期

6. **recurring_transactions表**: 循环交易
   - 支持daily, weekly, monthly, yearly频率
   - 自动化重复交易记录

#### 数据库优化
- 合理的索引设计，提高查询性能
- 外键约束确保数据完整性
- 分区表策略支持大量数据存储

## 3. 功能实现

### 3.1 用户认证系统
- **注册功能**: 用户名验证、邮箱确认、密码强度检查
- **登录功能**: JWT令牌生成、自动续期、安全登出
- **密码安全**: Bcrypt加密、安全存储、防暴力破解
- **会话管理**: 无状态认证、跨域安全处理

### 3.2 账户管理
- **多账户支持**: 支持银行账户、信用卡、现金等多种账户类型
- **余额跟踪**: 实时余额更新、交易关联
- **账户分类**: 清晰的账户类型管理

### 3.3 交易管理
- **交易录入**: 支持收入和支出两种类型
- **分类管理**: 自定义分类、颜色标识、图标选择
- **搜索筛选**: 按日期、金额、分类、账户筛选
- **批量操作**: 支持批量导入、编辑和删除

### 3.4 预算系统
- **预算设置**: 按分类设置月度或年度预算
- **进度跟踪**: 实时显示预算使用情况
- **预警机制**: 预算超支提醒和建议
- **历史分析**: 预算执行情况统计

### 3.5 数据可视化
- **收支趋势**: 月度收支变化趋势图
- **分类统计**: 支出分类占比饼图
- **预算进度**: 预算完成度条形图
- **财务概览**: 关键财务指标仪表板
